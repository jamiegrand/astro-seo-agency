# Pause Session

## Step 1: Gather Current State

Collect information about the current session:

### Active Work
- Current task/issue being worked on
- Files currently being modified
- Progress percentage

### Session Context
- When session started
- What was accomplished
- What's in progress
- What was planned but not started
- Astro docs consulted during session

### Open Items
- Uncommitted changes (check `git status`)
- Failing tests or build issues
- Questions that need answers

2. Start dev server: `npm run dev`
3. Read this file
4. Check git status
5. Continue from "What's Next"

- Missing context? Run `/start` for fresh priorities
- Need Astro help? Run `/astro-check docs "[topic]"`
